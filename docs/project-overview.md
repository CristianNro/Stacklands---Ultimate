\# Project Overview



\## Identity



This is a Unity game inspired by Stacklands.



It is based on:

\- cards represented as UI elements

\- drag and drop interactions

\- stack formation

\- recipe-based transformations

\- timed crafting

\- automatic continuation of crafting when the resulting stack remains valid



\## Main goal



The project should grow by strengthening the current mechanics and architecture, not by replacing them.



\## Core gameplay expectations



The following must remain true unless explicitly changed:

\- cards can exist alone or inside stacks

\- stacks matter mechanically, not only visually

\- recipes are evaluated from stack contents

\- crafting can be timed

\- crafted outputs can spawn with animation

\- valid repeated crafting can continue automatically

\- tags are an important scalability mechanism



\## Future direction



Likely future expansions:

\- more card types

\- richer recipes

\- stronger tag systems

\- more advanced batch crafting

\- board restrictions and placement zones

\- improved feedback and animation

\- market/shop systems

\- progression systems

\- survival/economy layers
